  • Por que contratar um arquiteto?

    O arquiteto é o profissional certo para transformar as suas necessidades e desejos em soluções espaciais, e construir ou reformar o seu imóvel. Segundo o Conselho de Arquitetura e Urbanismo, ele é o profissional técnico e legalmente autorizado a fazer o projeto arquitetônico, a exigência central para que sua edificação seja construída e pode também realizar e/ou coordenar os projetos complementares necessários à sua construção.

    Ao unir técnica, criatividade, funcionalidade e estética, o profissional pode deixar o seu espaço mais agradável, acolhedor, flexível, ambientalmente sustentável e adequado às suas necessidades e ao seu orçamento.

    Você também não corre o risco de comprar material a mais ou a menos: ele vai calcular a quantidade necessária e elaborar um orçamento para facilitar a sua programação financeira, evitando assim gastos desnecessários.

    O arquiteto também está preparado para gerenciar todos os fornecedores da obra, desde o fabricante de ferragem pronta à loja de papel de parede, e prestar ajuda técnica e estética na compra dos materiais.

    Fonte: CAU/BR

  • Quais são as etapas do projeto?

    Após o fechamento do contrato é realizada uma reunião com cliente para entender as ideias, sonhos, expectativas e conhecer o espaço para levantamento de medidas e definição do programa de necessidades.

    Então é desenvolvido o Estudo Preliminar, que é a primeira proposta para o projeto. Nesta fase pode conter plantas humanizadas, definição de layout e perspectivas em 3D (a apresentação de variações do projeto também pode ocorrer nessa etapa).

    Após aprovação do Projeto Preliminar inicia-se o desenvolvimento do Anteprojeto, que é a evolução do estudo preliminar. Nele são especificados os elementos necessários para a compreensão do projeto pelo cliente. Podem ser apresentadas plantas dos pavimentos, cortes esquemáticos, fachadas principais e especificação prévia dos principais acabamentos.

    Com tudo aprovado e definido, o próximo passo é o desenvolvimento do Projeto Executivo. Esta é a fase que demanda mais tempo e é onde se detalha minuciosamente todo o projeto para que a execução do mesmo seja fiel à apresentada ao cliente. O conteúdo entregue nesta etapa é o detalhamento final com todas as informações técnicas necessárias para a contratação dos serviços que atuarão na obra.

    Observação: Alguns projetos demandam a etapa de Projeto Legal, que é o projeto encaminhado para aprovação na prefeitura. Nestes casos, o projeto executivo só poderá ser desenvolvido após a aprovação deste.

  • Como funciona o projeto à distância?

    O projeto é realizado totalmente online. As reuniões poderão ser realizadas através de chamada de vídeo por aplicativo a ser definido.

    As etapas são semelhantes ao projeto presencial. Primeiro é realizada uma reunião para levantamento de dados. Depois, o cliente precisa apresentar fotos, vídeos e dimensões do ambiente para que seja possível entender o espaço. Então, na etapa de criação é desenvolvido o Estudo Preliminar, Anteprojeto e Projeto Executivo. Com tudo aprovado, o cliente receberá um e-mail com todos os detalhamentos desenvolvidos na etapa de criação e poderá começar a executar o projeto.

  • Quanto tempo leva para um projeto ser desenvolvido?

    Cada projeto é único e exige uma demanda diferenciada. Para um projeto simples calcula-se, em média, duas semanas para o Estudo Preliminar, duas a três semanas para o Anteprojeto e três a quatro semanas para o Projeto Executivo.

    Alguns prazos não dependem somente do arquiteto, o que pode acarretar atrasos na entrega de alguma das etapas. No contrato de cada projeto será especificado o tempo estimado para desenvolver o material de acordo com a particularidade do projeto.
